# Divvy Bike Share Analysis

## Overview
This project is a data analysis case study conducted using spreadsheets and R.
The analysis explores Divvy bike-share usage patterns to understand how casual
riders and annual members use the service differently.

The insights from this analysis aim to support data-driven marketing and business
strategy decisions.

## Business Question
How do casual riders and annual members use Divvy bikes differently?

## Data Source
The data used in this project comes from publicly available Divvy bike-share trip
data. The dataset contains ride-level information including ride duration, rider
type, bike type, and time-based attributes.

## Tools Used
- Spreadsheets (data cleaning and initial exploration)
- R (data analysis and visualization)
- R Markdown (report generation)

## Analysis Process
1. Data cleaning and preparation
2. Exploratory data analysis
3. Comparison of usage patterns between rider types
4. Visualization of trends
5. Interpretation of findings

## Key Insights
- Casual riders generally take longer trips than annual members
- Annual members show more consistent usage during weekdays
- Usage patterns vary by time of day and season between rider types

## Project Output
The final analysis is presented as an HTML report generated using R Markdown.


## Conclusion
This case study highlights clear behavioral differences between casual riders and
annual members. These findings can be used to design targeted strategies aimed at
converting casual riders into long-term subscribers.
